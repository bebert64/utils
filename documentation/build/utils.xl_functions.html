dump_data(data_table, headers_excluded=None, first_row=1, first_column=1)¶

Dumps data into an Excel worksheet.

The function should be used on an empty worksheet, as the data will be dumped at the beginning of the sheet without verifying if something is already there.

Parameters

  • data_table (DataTable) – A list of dict, with identical keys. The keys will be used as header for the columns on the first row, and the values will be dumped into successive rows.

  • headers_excluded (Optional[List[str]]) – A list of keys NOT to be dumped. Default is an empty list.

  • first_row (int) –

  • first_column (int) –

Return type

None

order_table(table, header, ascending=True)¶

Orders the rows of a table by field name.

Parameters

  • table (COMRetryObjectWrapper) – The table to be ordered. The table needs to have header in its first row.

  • header (str) – The header for the column used for ordering the rows.

  • ascending (bool) – If True, rows will be ordered in ascending order. If False, rows will be ordered in descending order. Default is True.

None

get_data_row(row, headers_included=None)¶

A dictionary containing the row values.

It is possible to filter on the columns to be included, both via the cols_included, ot the translation argument (more details provided in the Parameters section). If both arguments are provided, ValueError will be raised.

Parameters

  • row (int) – The number of the row which values we want.

  • headers_included (Optional[List[str]]) – List of columns to be included in the results. Default is None, in which case all columns will be included.

Return type

Dict[str, Union[int, str, None]]

Returns

  • A dictionary, which keys are the table header (or their translation

  • if the translation argument has been provided), and which values

  • are the ones found in the row.
